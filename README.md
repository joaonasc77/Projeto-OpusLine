# Projeto de Análise de Dados com Power BI - OpusLine

Esse projeto foi desenvolvido com base em um cenário fictício da empresa **OpusLine**, que atua no setor de cosméticos e beleza, com o objetivo de praticar e consolidar conhecimentos em análise de dados utilizando o Power BI, passando por todas as etapas do processo, desde a obtenção e tratamento dos dados até a construção de dashboards e geração de insights.

## Sobre o Projeto

A base de dados e algumas medidas já estavam disponíveis, então o trabalho foi direcionado para:

- Construção de gráficos e visuais adequados para diferentes tipos de análise  
- Criação de KPIs e indicadores relevantes para o negócio  
- Organização das informações em um relatório claro e funcional  
- Desenvolvimento de navegação entre páginas  
- Aplicação de filtros e parâmetros para tornar a análise mais dinâmica  

## Desenvolvimento

O trabalho começou com a etapa de tratamento de dados utilizando o Power Query. Nesse processo, foram realizadas limpezas, ajustes de tipos de dados, remoção de inconsistências e organização das tabelas. Depois disso, avancei para a modelagem e criação de cálculos com DAX. Aqui foi possível explorar melhor o potencial da ferramenta, utilizando medidas, colunas calculadas, variáveis e funções de tabela. Com a base pronta, parti para a construção do dashboard, focando em uma visualização clara, intuitiva e que realmente facilitasse a análise.

## Aprendizados com o Projeto

- A escolha do gráfico certo faz muita diferença na interpretação dos dados  
- Nem todo dado precisa virar gráfico. Saber o que destacar é essencial
- O uso de DAX permite análises muito mais flexíveis e inteligentes  

## Insights obtidos

A partir da análise dos dados, foi possível identificar alguns padrões interessantes:

- Existência de variações relevantes ao longo do tempo, indicando possíveis sazonalidades.
- Diferença de desempenho entre categorias/segmentos, mostrando onde estão os pontos fortes e fracos.
- Alguns indicadores que pareciam bons à primeira vista se mostraram menos relevantes quando analisados em conjunto com outras métricas.
- A importância de olhar os dados de forma cruzada (e não isolada), já que muitos comportamentos só aparecem quando combinamos variáveis.
